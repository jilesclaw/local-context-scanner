# Haystack Dummy Project

This is a synthetic codebase for testing local retrieval augmented generation (RAG).

## Structure
- `services/` - Business logic (Look for payment_service.py)
- `models/` - Data models (Look for user.py, product.py)
- `utils/` - Helper functions (Look for rate_limiter.py)
- `config/` - Configuration files (Look for app_config.json)
- `routes/` - API endpoints
- `tests/` - Unit tests

## Task
Add rate limiting to the payment service using the utility provided.
